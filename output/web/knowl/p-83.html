It would seem that by symmetry one ought to be able to show \(I(V(J)) = J\) holds for any radical ideal in the same manner that we proved \(V(I(W)) = W\) for any affine variety \(W\text{,}\) but this does not work as the previous example shows. What goes wrong? The symmetric proof does show that \(I(V(J)) = J\) for any ideal \(J\) that has the form \(I(W)\) for some \(W\), but it is not true that every radical ideal \(J\) is of this form for any field. Indeed, for \(J\) as in the previous example, \(J\) is a radical ideal and yet \(J \ne I(W)\) for any subset \(W\) of \(\mathbb{A}^n_{\mathbb{R}}\text{.}\)
